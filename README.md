
### 🧠 Description

UniMate is an AI-powered study assistant that helps students with:

* Explanations
* Quiz generation
* Study planning

All interactions happen inside Telegram for zero friction.

---

### 🛠 Tech Stack

* n8n (workflow automation)
* OpenRouter (LLM API)
* Telegram Bot API
* GitHub Pages (hosting)
